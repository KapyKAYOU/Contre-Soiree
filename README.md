# Contre-Soirée

Bot Discord pour la gestion de games communautaires (LG / UHC).

## Fonctionnalités
- Gestion de whitelist (add, remove, edit)
- Ajout multiple de pseudos
- Permissions par rôles (Host / Officier / Sergent)
- Commandes list et copy optimisées
- Backups automatiques

## À venir

- Annonces Twitch avec embed personnalisé
- Migration vers les slash commands (/)
- Hébergement VPS 24/7

- Python 3.10+
- discord.py
